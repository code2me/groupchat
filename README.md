# GroupChat App

Realtime chat app 

## Built with websockets using 
- Node.js, 
- Express and 
- Socket.io with 
- Vanilla JS 
on the frontend with a custom UI (Not Responsive yet)

# You can Have a look here [Website](https://groupchat007.herokuapp.com/)
